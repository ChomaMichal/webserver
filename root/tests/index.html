Browser test hub

This tree is meant to make server behavior visible directly in the browser: autoindex, redirects, upload permissions, and error pages.

autoindex-onDirectory listing should appear because no index file is present. autoindex-offThis route serves an index page instead of a listing. redirect-302This location returns 302 and lands on a target page. upload-allowedUse POST or DELETE against /uploads from the main dashboard. upload-deniedThis route should show upload disabled behavior. permission-403Directory permissions should block access here. errorsBrowse the named error pages here.
