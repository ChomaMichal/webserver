# Examples run instructions

The repository includes three example files to test POST requests.

- `examples/post_form.html`
	- Open this file in your browser to submit POST requests via a simple form.

- `examples/test_requests.sh`
	- Make executable and run:
		```bash
		chmod +x examples/test_requests.sh
		./examples/test_requests.sh
		```

- `examples/post_requests.js`
	- Use in a browser console: open the file and call the provided functions (e.g. `simplePost();`).
